shell变量

变量的类型

  • 自定义变量

    • 用户自己定义的变量
    • 变量没有数据类型，默认为字符串类型
    • 初始化的时候=两边都不要留空
    • 变量的调用需要加$
    • 当变量和其它字符串一起的时候，调用加{}
    • 不能直接变量=变量，需要在右边的变量加$
    • 没有分号，以换行作为一个语句的结束

  • 位置变量

    计算机把命令行参数的值传递给特定的变量从0开始调用，10以上的用{}括起来

  • 预定义变量

    计算机已经给一些特定的字符赋予了特殊的意义

    $*:所有的位置变量，但是不包括$0
    $@:所有的位置变量，但是不包括$0
    $#:位置变量的个数，但是不包括$0
    $?:获得之前(上一个)进程(shell命令)结束的状态码 (0 表示成功, 1 表示失败)
    $$:获得当前进程 ID
    $!:获得之前(上一个)进程 ID

  • 环境变量

    • 计算机已经赋值好的变量
    • env查看所有的环境变量
    • 调用环境变量也需要加$

只读变量：使用 readonly 命令可以将变量定义为只读变量，只读变量的值不能被改变。

删除变量：使用 unset 命令可以删除变量（不能删除只读变量）。

变量运算

  • $(())

    • 能进行+-*/%运算
    • 能进行自加自减
    • 能进行幂数运算
    • 不能进行浮点型运算

  • $[]

    • 能进行+-*/%运算
    • 能进行自加自减
    • 能进行幂数运算
    • 不能进行浮点型运算

  • expr

    • 需要用命令置换符``置换出来结果
    • 能进行+-*/%，但是在进行乘法运算的时候需要加\*和通配符*区分
    • 不能进行幂数运算
    • 不能进行浮点型运算

shell字符串

字符串是shell编程中最常用最有用的数据类型，字符串可以用单引号，也可以用双引号，也可以不用引号（有空格时一定要用引号）。

单引号字符串的限制：

  • 单引号里的任何字符都会原样输出，单引号字符串中的变量是无效的。如果要在单引号里使用变量，需要将变量再用一对单引号包括起来。
  • 单引号字串中不能出现单独一个的单引号（对单引号使用转义符后也不行），但可成对出现，作为字符串拼接使用。

双引号的优点：

  • 双引号里可以有变量
  • 双引号里可以出现转义字符
  • 双引号里可以直接使用单引号，不用转义
